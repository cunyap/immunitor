Please note this is only a draft of our final project, data are currently not being collected, stored or analyzed.

Immunitor is a data repository platform, yes, another one! There are so many awesome self-reporting tools being developed, however, being biotechnology scientists, we saw a need that was still unmet: centralized data collection, data validation and automated data processing.

During the CodeVsCOVID19 Hackathon we prototyped a platform where patients that get tested can submit their result and being in full control of how much additional information they want to provide. However, the first question we ask is "Do you have a test serial number?". We want to collect data as anonymously as possible, but we also want to be sure that the entries are valid, that they don't get over counted, and that are compatible with bulk uploads (in case hospitals also want to share data on Immunitor).

Using the test serial number allows:

  • to be completely anonymous: we don't need to know the name, the email or anything at all from the user.
  • to tag the data entry as validated.
  • to retrieve information about the type of test (PCR based vs antibody based).
  • to merge data from different cantons, manufacturers, clinics.
  • to collect the data that will come from home testing, that would otherwise be lost.
  • to have a well-organized database (on OpenBis) that can be accessed by researchers and authorities.

Additionally, we allow for subsequent entries, by providing a QR code that the user can store and use again to modify previous entries (for example if you don't know what the test serial number is, and then you find out and want to validate your data) and, most importantly, add subsequent entries that maintain a temporal relationship (for example if you repeat the test and change your status from infected to immune). Last but not least, we provide a neat **live data visualization* page, where anyone can see how their contribution is helping to fight the Covid19 crisis.

Open source

Our platform is fully open source and can be view or used by anyone. External auditing of the platform allows to increase the trust in the repository showing that the data is kept save and anonymous.

Encrypted

Your data is transmitted trough SSL.

Your data in Switzerland

Your data is kept on servers hosted in Switzerland. This demo server is actually hosted on ETHZ servers.

Quick setup

The platform is very easy to setup, maintain, extend and use.
